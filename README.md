# CBR Currency Parser

Парсер курсов валют Центрального банка Российской Федерации (ЦБ РФ). Скрипт извлекает актуальные курсы CNY (юань), USD (доллар США) и EUR (евро) за две последние даты с официального сайта ЦБ РФ и генерирует HTML-страницу с таблицей данных.

## Функциональность

- **Парсинг данных**: Автоматическое извлечение курсов валют с сайта cbr.ru с использованием BeautifulSoup.
- **Вывод в консоль**: Отображение курсов в терминале для быстрого просмотра.
- **Генерация HTML**: Создание статической HTML-страницы `rates.html` с таблицей курсов для удобного просмотра в браузере.
- **Обработка дат**: Динамическое получение дат из страницы ЦБ РФ (обычно позавчера и вчера).

## Требования

- Python 3.6+
- Библиотеки:
  - `requests` (для HTTP-запросов)
  - `beautifulsoup4` (для парсинга HTML)
  - `lxml` (парсер для BeautifulSoup, рекомендуется для лучшей производительности)

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/cbr-currency-parser.git
   cd cbr-currency-parser
   ```

2. Установите зависимости:
   ```bash
   pip install requests beautifulsoup4 lxml
   ```

## Использование

Запустите скрипт `main.py`:
```bash
python main.py
```

Скрипт выполнит следующие действия:
- Получит данные с сайта ЦБ РФ.
- Выведет курсы в консоль.
- Создаст файл `rates.html` с таблицей курсов.

Откройте `rates.html` в браузере для просмотра результатов.

## Структура проекта

- `main.py`: Основной скрипт парсера.
- `rates.html`: Сгенерированная HTML-страница с курсами (создается при запуске).
- `README.md`: Документация проекта.

## Зависимости от сайта

Скрипт зависит от структуры HTML сайта cbr.ru. Если ЦБ РФ изменит разметку страницы, может потребоваться обновление селекторов в коде.
